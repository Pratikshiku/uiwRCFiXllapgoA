### 3. Feature Description
The system is divided into (roles): administrators, warehouse managers (the system has role management and can expand roles as needed).

**Warehouse Manager User Features:**
1. Login
2. Inventory Query
3. Purchase Inbound
4. Purchase Document Query
5. Current Inventory Query


**Administrator Features:**
1. Login
2. User Management
3. Role Management
4. Menu Management
5. Log Operations
6. Supplier Management
7. Customer Management
8. Product Management
9. Purchase Inbound
10. Purchase Document Query
11. Current Inventory Query
12. Sales Outbound
13. Sales Document Query
